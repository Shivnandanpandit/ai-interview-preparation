# AI Interview Preparation

## Overview
An AI-powered interview preparation platform that analyzes resumes and job descriptions to generate personalized interview strategies.

## Features
- Resume Analysis
- Job Description Matching
- Interview Question Generation
- Skill Gap Analysis
- Preparation Plan
- Resume PDF Generation

## Tech Stack
- React
- Vite
- Node.js
- Express.js
- MongoDB
- Gemini AI
- Puppeteer

## Installation

### Backend
```bash
cd Backend
npm install
npm run dev
```

### Frontend
```bash
cd Frontend
npm install
npm run dev
```
